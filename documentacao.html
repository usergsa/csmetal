📱 Guia de Uso

🖼️ Layout do App

  • Lista Horizontal de Imagens:
    Na tela inicial, as imagens são exibidas em uma faixa horizontal com rolagem.
  • Campo de Busca:
    Acima da lista, há um campo de busca centralizado, ao digitar, um filtro vertical é ativado.
  • Filtro Vertical:
    O filtro exibe os itens encontrados em uma lista vertical, facilitando a navegação por código ou nome.
  • Visualização em Tela Cheia:
    Ao tocar em uma imagem do filtro, ela é ampliada para tela cheia, permitindo edição do nome (em desenvolvimento).

🔧 Pré-requisitos

  • Conceda permissões para câmera e galeria ao iniciar o app.
  • Este app funciona com captura e gerenciamento de imagens.

📋 Menu Inicial

  • 📷 Capturar Fotos: Abre a câmera e permite capturar múltiplas imagens.
  • 📁 Carregar Fotos: Seleciona imagens da galeria do dispositivo.
  • 📄 Lista: Mostra todas as fotos classificadas.
  • 🗄️ Gerar Db: Gera códigos em massa para identificação de itens.
  • ↕️ Classificar: Ordena as imagens por código.
  • 📤 Upload: Envia imagens para o Servidor automaticamente.

🔎 Pesquisa e Comandos

Use o campo Pesquisar para:

  • 🔍 Buscar por código ou parte dele.
  • Cliente: ativar modo para adicionar cliente.
  • Etapa: ativar modo para adicionar nova etapa.
  • Upload: alternar upload automático ON/OFF.
  • Cup: ocultar botão de upload manual.
  • Eup: exibir botão de upload manual.
  • Kit: ativar modo kit para agrupar itens.
  • Ckit: desativar modo kit.
  • Zbl: recarregar fotos da galeria.
  • Zbdb: mostrar banco de dados.
  • Cmn / Smn: mudar visualização do menu (compacto/expandido).
  • Comp: compartilhar fotos.

📸 Captura e Classificação

  • Fotos capturadas são salvas com código Captura.
  • Fotos da galeria recebem código Galeria.
  • Você pode renomear o código de uma imagem para classificá-la.
  • Modo Kit permite adicionar itens auxiliares a uma imagem.

☁️ Upload

  • Uploads automáticos ocorrem ao adicionar novas fotos.
  • Botão UP (no canto inferior direito) permite envio manual dos pendentes.
  • As imagens enviadas são marcadas com uploaded: true.

📊 Relatório

  • O botão "📄 Lista" mostra agrupamento por código com quantidades.
  • Toque em um código para visualizar as imagens relacionadas.

✅ Dicas

  • Use o botão ↕️ Classificar para organizar os códigos.
  • Você pode excluir fotos, etapas ou dados segurando sobre os itens.

Este app salva seus dados localmente e integra com o Servidor para upload.
